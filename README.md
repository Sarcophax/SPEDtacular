## Project Overview

**SPEDtacular** is an online Computer-Aided Instruction (CAI) application. The system was designed specifically for special education students, with the goal of improving learning engagement, accessibility, and academic outcomes through interactive digital instruction.

The platform delivers **game-based learning experiences** focused on:

* Foundational **mathematics operations**
* Core **English grammar skills**

To support diverse learning needs, the system dynamically adapts lesson difficulty based on each learner’s performance.

---

## Key Features

### Interactive Educational Games

* Math operations (addition, subtraction, etc.)
* English grammar and language exercises
* Visual and interaction-based learning activities

### Dynamic Difficulty Adjustment (DDA)

* Automatically modifies content difficulty
* Responds to real-time student performance
* Helps prevent frustration and boredom
* Promotes individualized learning paths

### Teacher and Student Management

* Separate data handling for teachers and students
* Persistent user profiles
* Performance and engagement tracking

### Cloud-Based Backend

* Real-time data storage and retrieval
* Secure authentication and session handling
